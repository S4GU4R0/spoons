# spoons
My Hammerspoon spoons

## Rabbit Hole
Whenever you use cmd W to close a tab/window, it let's you leave a note and save it to a CSV.

### How it works
It'll pull up the `chooser` which looks like Spotlight's little pop up input. `Esc` cancels, `Return` submits. Allows for empty input. One browsers and in Finder, it'll capture the url. Elsewhere, it captures the window/tab title.
